# Alinta Coding Challenge
This is my code for Alinta's coding test.


### Assumptions

All Customer's fields are required.

### How to run

#### Build app
F5 in Visual Studio, or:
```
dotnet build
```

#### Run app
F5 in Visual Studio, or
```
dotnet run --project AlintaCodingTest.Api
```

#### Run tests
Use Test Explorer in Visual Studio, or
```
dotnet test
```
